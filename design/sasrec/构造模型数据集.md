# SASRec 离线数据集

## 目录约定

```text
scripts/sasrec/build_dataset.py
data/processed/<dataset>/interactions_5core.tsv
data/processed/<dataset>/sasrec/
```

## 构造边界

离线脚本负责：

1. 按用户、时间和商品排序；
2. 将 raw item ID 映射为从 1 开始的连续 ID，0 保留为 PAD；
3. 使用 leave-one-out 构造 Train、Valid 和 Test；
4. 生成定长训练输入、正样本和固定负样本；
5. 为 Valid/Test 每个用户固定采样 100 个未交互负样本。

训练代码只校验配置并 mmap 读取以下产物：

```text
train_sequences.npy
train_positives.npy
train_negatives.npy
valid_histories.npy
valid_candidates.npy
test_histories.npy
test_candidates.npy
dataset_config.json
dataset_stats.json
```

## 命令

```bash
python -u scripts/sasrec/build_dataset.py --dataset ml-1m
python -u scripts/sasrec/build_dataset.py --dataset amazon-beauty
python -u scripts/sasrec/build_dataset.py --dataset amazon-books
```
